# Hirsch-Fye-QMC
Hirsch-Fye QMC Solver

## Required libraries 
+ FFTW3
+ OpenBLAS


Build Reciepe
--------------
    python3 ./setup.py build # It will automatically download and install libraries.
    python3 ./main.py # Run simulations
